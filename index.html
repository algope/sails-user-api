Sails User API

A SailsJS basic API for Create, Authenticate and Authorize users

View on GitHub Download .zip Download .tar.gz

About

A SailsJS API for Create, Authenticate and Authorize users on your server. It uses the JSONWebToken.

Features

  • Token based Authentication.
  • Authorization via Policies.
  • MongoDB ready.
